# インストールとビルド

```
$ npm install
$ npx spago install
$ npm run build
```

# アプリの開き方

```
$ npm run serve
```

または、

`public/index.html`を静的なファイルとしてブラウザ(Chrome, Firefoxなど)で開きます。

# テスト

$ npm test
